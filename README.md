# InstantRamen
InstantRamen es un caótico party game para hasta 4 jugadores. Tu misión: conseguir los fideos y combinar los ingredientes exactos para servir el tazón definitivo antes de que se acabe el tiempo... o tus amigos te arruinen la receta. Rápido, adictivo y la fórmula perfecta para romper amistades cocinando. ¿Quién será el chef supremo?
